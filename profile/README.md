# Pomfort Silverstack DIT Tool

**Pomfort Silverstack DIT Tool** is the industry-standard media management and backup application for digital imaging technicians on film and television productions. As a powerful **Pomfort Silverstack Offloader**, it performs parallel offload from multiple cards simultaneously with hardware-accelerated checksum verification (xxHash, MD5, SHA-256). This **Pomfort Silverstack Metadata Editor** allows you to add look-up tables (LUTs), color decision lists (CDLs), ALE metadata, custom tags, and scene/shot information to clips. With **Pomfort Silverstack Report Generator**, you can create detailed PDF reports including camera logs, waveform scopes, thumbnails, and quality control notes for post-production teams.

## Features

- **Media Backup**: The **Pomfort Silverstack DIT Tool** copies to multiple destinations with verification.
- **Card Offload**: Use **Pomfort Silverstack Offloader** to offload up to 6 memory cards at once.
- **Metadata Management**: As a **Pomfort Silverstack Metadata Editor**, it supports ARRI, RED, Sony, and Canon.
- **Documentation**: This **Pomfort Silverstack Report Generator** exports CSV, HTML, and PDF formats.
